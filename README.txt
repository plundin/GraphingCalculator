
Desktop Graphing Calculator
------------------------------------------------------------------------------------------
Author: Patrik Lundin, patrik (at) lundin.info
http://www.lundin.info

Desktop calculator with function graphing capabilities. 

The calculator handles the normal operations plus the ability to plot
simple functions (only functions of type f(x) in a standard coordinate system).

See website http://lundin.info/calculator for more information.


Install Instructions
------------------------------------------------------------------------------------------
To install extract the zip file and click the setup.exe file 

Visual Studio Solution
------------------------------------------------------------------------------------------
The complete visual studio solution can be found in the "Visual Studio Solution" folder,
open the GraphingCalculator.sln from within Visual Studio.
